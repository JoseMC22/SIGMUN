# SIGMUN Project Instructions

## Estándares del Proyecto

Todas las actividades de desarrollo deben seguir los estándares definidos en el skill `@sigmun-standard`.

### Aplicar automáticamente:
- Invocar `@sigmun-standard` para todas las nuevas funcionalidades
- Aplicar las prioridades de enforcement en orden:
  1. Seguridad
  2. Contrato API
  3. Tipado/Lint
  4. Validación
  5. Testing
  6. Observabilidad
  7. Guías visuales/UX

### Contexto del Proyecto:
- **Monorepo:** Backend (NestJS) + Frontend (Next.js)
- **Package Manager:** pnpm >=8.0.0
- **Database:** SQL Server con driver `mssql`
- **Design System:** SAT ICA (colores, tipografía, accesibilidad)

### Cuando usar @sigmun-standard:
- ✅ Crear nuevos archivos TypeScript/React
- ✅ Revisar arquitectura y estructura
- ✅ Diseñar componentes UI/UX
- ✅ Implementar autenticación y seguridad
- ✅ Configurar validación de datos
- ✅ Establecer logging y observabilidad
- ✅ Revisar código antes de mergear
